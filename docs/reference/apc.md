# apc S3 class

Class for (Bayesian) age-period-cohort objects

## Usage

``` r
apc()
```

## Value

apc class

## Details

[`bamp`](https://volkerschmid.github.io/bamp/reference/bamp.md) will
return an object of class apc. Available functions are

- [`plot.apc`](https://volkerschmid.github.io/bamp/reference/plot.apc.md)
  plots main effects

- [`print.apc`](https://volkerschmid.github.io/bamp/reference/print.apc.md)
  print summary of model and effects

- [`effects.apc`](https://volkerschmid.github.io/bamp/reference/effects.apc.md)
  extract effects (mean, median and quantiles)
